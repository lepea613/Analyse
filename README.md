# Analyse

## Overview
Dieses Proekt dient zur Datenanalyse verschiedener Fitnesswerte, welche mittels Fitbittrackern erhoben wurden. Bevorzugt in Python programmiert. 

Ziel ist eine Veranschaulichung der Daten, sowie dem Vergleich einzelner Werte.

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Data visualization
- Basic statistical analysis
- Implementation using Python tools

---

## Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
